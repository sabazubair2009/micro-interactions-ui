# MicroUI - Button System Lab 🎛️

A modern UI practice project built using **HTML + Tailwind CSS**, focusing on micro-interactions, hover effects, and smooth transitions for buttons.

---

## 🚀 Project Overview

MicroUI is a small frontend project that demonstrates different types of buttons with interactive hover effects. It helps in understanding how transitions, shadows, rotation, and color changes work in real UI components.

---

## 🎯 Features

- Responsive navigation bar
- Clean dark-themed UI
- 3 button styles:
  - Primary Button
  - Secondary Button
  - Outline Button
- Smooth hover animations
- Shadow effects and rotations
- Color transitions using Tailwind CSS

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS (CDN)

---

## 🎨 Button Effects Included

- Hover rotation
- Shadow glow effects
- Border color transitions
- Background color changes
- Smooth animations using `transition` and `ease-in-out`

---

## 📸 UI Sections

- Navbar with brand "MicroUI"
- Button showcase cards
- Micro interaction demo section

---

This project was created to practice:
